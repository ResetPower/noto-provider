# noto-provider

Provide part of Noto font family for web applications.

## Provided Fonts

- Noto Sans JP
- Noto Sans SC

## Provided Weights

- Thin 100
- Light 300
- Regular 400
- Medium 500
- Bold 700
- Black 900

## Usage

In case your bundler supports importing css file, use

```javascript
import "noto-provider";
```
